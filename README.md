# DS 3026 Final Project: Predicting and Analyzing Rice Paddy Yield

## Overview
This repository contains the full analysis for our final project in DS 3026 - Inference and Prediction. 
We analyze a dataset of 2,798 rice paddy crop records from Tamil Nadu, India to answer both an 
inferential question (does rice variety significantly affect yield?) and a predictive question 
(can we accurately predict yield given farm inputs and weather conditions?).

## Dataset
The dataset contains farm inputs, weather data, and paddy yield (kg) for plots across Tamil Nadu, India.  
Source: [UCI Machine Learning Repository](https://archive.ics.uci.edu/dataset/1186/paddy+dataset)
